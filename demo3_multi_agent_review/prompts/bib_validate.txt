Run the /validate-bib skill. It will:

  1. Extract every citation key in Bibliography_draft.bib (or
     Bibliography_base.bib).
  2. Scan Paper/main.tex and Paper/sections/*.tex for every \cite, \citet,
     \citep, \citeauthor, \citeyear key.
  3. Report:
     - Missing entries (cited but not in bib) — CRITICAL
     - Unused entries (in bib, never cited) — informational
     - Potential typos (similar but non-matching keys)
     - Malformed bib entries

The skill is repository-aware. No arguments needed beyond the trigger.
